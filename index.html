Frequently Asked Questions

First of all, DO NOT ASK QUESTIONS ABOUT THIS LAUNCHER IN THE OFFICIAL PRISM LAUNCHER GITHUB OR DISCORD.
If you'll ask a question mentioned down there on our Discord server (and i hope not on GitHub Issues), you'll be ignored, ridiculed, or even muted
  1. How do I download PineconeMC???
    You can download it via site's download page (alternative),
    or you can via Github Releases page.
    If you're using Windows, you'll probably need PineconeMC-Windows-MSVC-Setup-*.exe.
  2. Is it ElyPrismLauncher? What is this?
    See gray text under PineconeMC logo above these blocks.
  3. Is PineconeMC free? Why it launches demo version?
    You have to create a new account. It can be either Ely.by or offline one.
  4. Is PineconeMC safe? Does it have any malwares?
    It is safe, if you're downloading it from official sources, such as:
    • pineconemc.ru
    • elyprismlauncher.github.io
    • ElyPrismLauncher/Launcher
    And you still can review and build launcher's source code by yourself.
  5. Why is my game not downloading? Why game downloading stuck at [insert random percent here]%?
    Your ISP may block needed domains for downloading. Try using a VPN and try downloading again.
  6. What exactly is Ely.by?
    It's an account and skin system which aims to replace official Minecraft's one.
    Unfortunately, it doesn't support capes while official system does, in this area you have to use mods.
    For more information about Ely.by, read their FAQ.
  7. Which account type should I use to have my own skin? Ely.by or Offline?
    Offline accounts can't have skins, unless you're playing on a server/modded with plugins/mods which can change your skin.
    You should use an Ely.by account if you want to see your skin everywhere you play.
  8. What servers can I join with an Ely.by/Offline account?
    Servers with online-mode=false.
  9. What's the minimal required version of Windows for PineconeMC?
    It's Windows 10. If you're using something lower, consider using Legacy Launcher (lightweight Ely.by launcher which only launches the game, supports Windows 7+) or try linux lol
  10. Windows says "This app can't run pn your pc", what do I do?
    You may have downloaded a version for other architecture than yours, recheck things.
  11. Can PineconeMC download modpacks, mods, shaders,...?
    Yes it can. PineconeMC is a fork of PrismLauncher, haven't you noticed?
