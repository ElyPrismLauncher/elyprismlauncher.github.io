Frequently Asked Questions

First of all, DO NOT ASK QUESTIONS ABOUT THIS LAUNCHER IN THE OFFICIAL PRISM LAUNCHER GITHUB OR DISCORD.
If you'll ask a question mentioned down there on our Discord server (and i hope not on GitHub Issues), you'll be ignored.
  1. Is it ElyPrismLauncher? What is this?
    See gray text under PineconeMC logo above these blocks.
  2. Is PineconeMC free? Why it launches demo version?
    You have to create a new account. It can be either Ely.by or offline one.
  3. Is PineconeMC safe? Does it have malware?
    It is safe, if you're downloading it from official sources, such as:
    • pineconemc.ru
    • elyprismlauncher.github.io
    • ElyPrismLauncher/Launcher
    And you still can review and build launcher's source code by yourself.
  4. Why is my game not downloading? Why game downloading stuck at [insert random percent here]%?
    Your country or ISP may block needed download servers. Try using a VPN.
  5. How do I move entire launcher to another disk?
    Use portable builds, they include all dependencies and instances in one folder, which you can move to another disk.
  6. What exactly is Ely.by?
    It's an account and skin system which aims to replace official Minecraft's one.
    It doesn't support capes while official system does, in this area you have to use mods.
    For more information about Ely.by, read their site and FAQ.
  7. What servers can I join with an Ely.by/Offline account?
    Servers with online-mode=false.
  8. What's the minimal required version of Windows for PineconeMC?
    It's Windows 10. If you're using something lower, consider using Legacy Launcher (lightweight Ely.by launcher which only launches the game, supports Windows 7+) or try linux lol
  9. Windows says "This app can't run on your PC", what do I do?
    You may have downloaded a version for other architecture than yours, recheck things.
  10. Can PineconeMC download modpacks, mods, shaders,...?
    Yes it can. PineconeMC is a fork of PrismLauncher, haven't you noticed?
